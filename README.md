# wuster_cv
武汉科技大学个人简历

参考自一个祖传的模板，以及https://github.com/LeyuDame/BNUCV/tree/main 上的BNU的latex简历模板中的代码。
		注意要用XeLaTeX编译链进行编译，且要进行三次编译才能显示照片。
		vscode+latex的话，配置的json中"latex-workshop.latex.recipes"添加：
		{
            "name": "XeLaTeX*3",
            "tools": [
                "xelatex",
                "xelatex",
                "xelatex"
            ]
        }
		即可使用三次XeLaTeX编译。
